Weather App

A clean, interactive weather dashboard widget that displays live weather updates and forecasts using the [Open-Meteo API](https://open-meteo.com/). Built with HTML, CSS, and JavaScript.

-Real-time weather info (temperature, wind, humidity, pressure)
-Daily forecast and custom date picker
-Weekly temperature chart powered by Chart.js
-Modern, responsive UI with Font Awesome icons
-No API key or server needed — works straight from your browser
